Initialize

Essa função é executada sempre que o arquivo JavaScript inicia ou é utilizado dentro de outro JavaScript. Observe o console de desenvolvedor.



After Render

Esse data-bind é muito útil quando queremos executar algum código apenas depois do componente ser renderizado! O alerta exibido ao carregar essa página ocorre por causa de um afterRender!
Dica: podemos passar parâmetros para a função. Exemplo: data-bind="afterRender: () => myFunction('myParameter')"



Loader

Esse é um data-bind dos mais úteis: criamos um efeito de Loading nos componentes que desejamos. E o melhor: é muito simples! Precisamos apenas definir uma variável que indica se o Loader está ativo e atrelá-la à alguma Tag HTML. Observe o exemplo abaixo.
